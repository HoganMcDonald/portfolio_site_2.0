I Know When You're Going to Die

I placed 2nd competing alongside a designer in the 2017 Hack for Change Hackathon as part of Twin Cities Startup Week. Our app leveraged the event’s sponsor’s (Nexosis) machine learning api as well as CDC data on mortality to predict a users exact date of death. The project required us to clean a dataset, upload that set to Nexosis using AWS, and train a regression model with 3 inputs and a single output feature.
